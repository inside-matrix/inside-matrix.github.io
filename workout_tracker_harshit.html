Export

📊 Export to Excel (CSV)

Downloads two CSV files that match your Excel tracker sheets exactly — open them in Excel and paste into Strength Log / Cardio Log.

🔐 Full Backup (JSON)

Exports all sessions as JSON — use this to back up and restore your data, or switch devices.

Import

📂 Restore from JSON backup

Merges sessions from the JSON backup into your current data. Duplicate sessions (same ID) are skipped.

📋 Import from CSV

Import a Strength Log CSV (exported from this app or pasted from Excel). Each row becomes a logged set. All rows with the same Date + Workout are grouped into one session.

Danger Zone

Export a backup before clearing. This cannot be undone.

About Storage

Data is stored in your browser's localStorage for this device.
• Export JSON regularly as a backup.
• To move to another device: Export JSON → open this file in the new browser → Import JSON.
• To sync with Excel: Export CSV → paste into your Excel tracker's Strength Log sheet.
• Clearing browser data or private/incognito mode will erase localStorage.
